## 📦 Tech Stack

- ⚛️ **React** — Frontend UI
- 🎯 **Appwrite** — Backend as a Service (Database, API)
- 🌐 **Vercel** — Deployment platform
- 💅 **CSS** — Styling
- 🧪 **React Hooks** — State and effect management
- 🧠 **Appwrite Query** — For filtering trending movies
